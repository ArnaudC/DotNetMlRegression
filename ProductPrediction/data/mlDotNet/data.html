Données

La base de données Open Food Facts est disponible sous la licence Open Database License.
Les contenus individuels de la base de données sont disponibles sour la licence Database Contents License.
Les photos de produits sont disponibles sous la licence Creative Commons Attribution Partage à l'identique. Elles peuvent contenir des éléments graphiques soumis à droit d'auteur ou copyright, qui peuvent dans certains cas être reproduits (droit de citation ou "fair use").

Merci de prendre connaissance des conditions d'utilisation et de réutilisation des données avant de les réutiliser.

Nous sommes très intéressés de savoir à quoi servent les données d'Open Food Facts, ce n'est pas obligatoire, mais nous serions ravis que vous nous informiez de vos réutilisations afin que nous puissions les partager avec la communauté Open Food Facts.

Export des données d'Open Food Facts

Tous les dumps et exports sont générés chaque nuit.

Pour le dump MongoDB et l'export CSV, la description des champs est disponible sur https://world.openfoodfacts.org/data/data-fields.txt

Dump MongoDB

Les données de tous les produits sont disponibles dans un dump de base de données MongoDB.

Lien : https://world.openfoodfacts.org/data/openfoodfacts-mongodbdump.tar.gz

Export CSV des données

Les données de tous les produits, ou d'une partie des produits, peuvent être téléchargées au format CSV (utilisable dans OpenOffice, Excel et beaucoup d'autres logiciels) via le formulaire de recherche.

Lien : https://fr.openfoodfacts.org/data/fr.openfoodfacts.org.products.csv

L'encodage du fichier est l'Unicode UTF-8. Le caractère de séparation des champs est (tabulation).

Export RDF des données

La base de données est également disponible sous format RDF. Voir l'annonce sur le blog.

Lien : https://fr.openfoodfacts.org/data/fr.openfoodfacts.org.products.rdf

API JSON expérimentale

Une API JSON est également disponible pour consulter les informations d'un produit. Cette API est notamment utilisée dans l'application mobile Open Food Facts pour iPhone et Android.

Cette API est expérimentale et peut être amenée à changer, merci de nous signaler si vous l'utilisez, afin que l'on puisse vous prévenir des éventuels changements.

URL de consultation : https://fr.openfoodfacts.org/api/v0/produit/barre.json

Exemple : https://fr.openfoodfacts.org/api/v0/produit/3029330003533.json

Sur le wiki : début de documentation de l'API en lecture et écriture.

Application mobile Android et iPhone

Le code de l'application mobile Open Food Facts est disponible sur github. L'application permet de scanner le code barre d'un produit, d'afficher les informations et d'envoyer des photos et informations sur les produits manquants.

L'application est développée en HTML et Javascript grâce à Phonegap (Cordova) qui permet d'utiliser le même code pour toutes les plateformes mobiles.

Liste de discussion données, API et exports

Vous pouvez également rejoindre la liste de discussion (en anglais) pour échanger au sujet des données, de l'API et des exports, présenter vos réutilisations, et être averti des améliorations et changements. Pour vous inscrire, envoyez un e-mail vide à api-subscribe@openfoodfacts.org

Contact

Pour toutes questions sur les données, vous pouvez nous écrire à contact@openfoodfacts.org

En particulier nous pouvons vous inviter sur notre outil Slack de discussion entre développeurs, contributeurs et réutilisateurs.
